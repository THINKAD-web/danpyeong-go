## 기능 제안

### 1. 도형/각도 문항에 실제 그림 필요
- 현황: "오른쪽 그림과 같이"라는 표현이 생성되는데 실제 그림이 없음
- 제안: 도형·각도 단원 문항 생성 시 SVG/이미지 첨부 기능 추가
- 관련: 도형 단원 AI 생성 제약 강화도 함께 검토

### 2. 난이도 '최상' 옵션 추가
- 현황: 하/중/상 3단계 제공. 교사 의견으로 '상'이 다소 쉬움
- 제안: VERY_HARD (최상) 옵션 추가 — 복합 문장제·창의적 사고 수준
- 영향 범위: GenerateInputSchema, 프롬프트 diffLabel, UI select

### 3. 학생 응시 코드 단순화
- 현황: shareToken이 길고 복잡해 학생이 직접 입력하기 어려움
- 제안: 6자리 숫자 코드(별도 테이블)를 shareToken 매핑으로 추가 제공
  - 예) 코드 `482931` → /play?token=xxxxx 리다이렉트
- 고려사항: 코드 충돌 방지, 만료 정책

### 4. 교사 간 문항 공유 / 커뮤니티
- 제안: 교사가 자신의 문항을 공개 설정 → 다른 교사가 가져다 쓸 수 있는 공유 풀
- 추가 아이디어: 포인트/기여 시스템, 좋아요/북마크
- 장기 로드맵 항목. 공개 연습 방향과 연결 검토 필요
